# Power Omni

O [Power Omni] é uma solução de IA corporativa que usa o Microsoft Foundry, permitindo que empresas possam trazer a IA generativa para dentro de suas empresas.  
Principais recursos:
- Os dados ficam no seu tentant do Azure 
- Você controla os custos de uso 
- Você pode integrar a IA com diversas fontes de dados da sua empresa (arquivos, bancos SQL, sharepoint, logic apps, fabric, APIs próprias)

Este repositório contém os exemplos em código úteis para usar com o [Power Omni].   
Se você está procurando por documentação de uso, veja a doc oficial em https://docs.poweromni.ai!

Aqui você vai encontrar os recursos que precisa se você está tentando customizar o seu Power Omni, como por exemplo, conectar com suas APIs, bancos de dados, etc.

# Integrações  

As integrações são conexões customizadas construídas usando os recursos nativos do Power Omni (como os conectores, etc.)
O [diretorio integrações](integracoes/) contém os exemplos e instruções que podem te ajudar implementar no seu ambiente.

> [!TIP]
> Qual a diferença entre uma Integração e um Conector?
> Um conector é algo nativo no Power Omni, onde você pode criar e configurar a conexão com a fonte externa
> Uma integração é o processo de usar um conector para fazer seu agente se comunicar com algum dado externo.  
> Por exemplo, o Power Omni tem o conector de OpenAPI, que conecta com API HTTP.  
> A integração de sql, usa o conector de OpenAPI para que você possa conectar com seu SQL Server.
