# Power Omni

Este repositório contém os exemplos em código úteis para usar com o [Power Omni](https://poweromni.ai)  
Se você está procurando por documentação de uso, veja a doc oficial em https://docs.poweromni.ai!

# Conectores  

Os conectores do Power Omni permitem que você integrar a IA corporativa aos seus dados!  
O [diretório de conectores](conectores/) contém os exemplos de código que você pode usar para diferentes fontes.  

# Ajuda

A ideia desse repositório é prover códigos e exemplos que ajudem você e seu time a configurar o Power Omni por conta própria.  
Mas entendemos que podem existir dúvidas ou dificuldades e podemos te ajudar mais rápido com isso.  

Se você já é cliente e tem o Power Omni instalado, procure o canal oficial de comunicação e nosso time irá te ajudar o mais breve possível.  

Se você ainda não é cliente, [agende uma reunião com nosso time comercial](https://poweromni.ai/apresentacao) para contratar uma opção de suporte.
