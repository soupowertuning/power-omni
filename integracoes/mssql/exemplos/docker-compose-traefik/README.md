# Usando com docker e traefik

Este diretório contém um exemplo de como usar docker compose e traefik (que facilita o uso com https).  

Pré-requisitos:

- ambiente linux com docker instalado 
- Liberar os ips de internet para a porta HTTPS e HTTP
- Apontar um nome de DNS para o IP da máquina (nome que você vai querer usar para acessar)
