# Schemas OpenAPI

Para que o Power Omni funciona corretamente com o OpenAPI, é preciso que o schema especificado esteja em um formato correto.  
A maior parte do schema segue o padrão OpenAPI e você pode encontrar mais detalhes aqui: https://swagger.io/specification/

Os arquivos desse diretório contém alguns exemplos práticos extraídos de agentes reais que funcionaram (com alterações de alguns nomes e textos para explicação).
